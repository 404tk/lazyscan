# LazyScan  

本项目基于[fscan](https://github.com/shadow1ng/fscan)进行拓展，仅供学习交流，请勿非法利用。  
功能概述：在`PoC`验证的基础上实现了`Exploit`利用  

## 常见服务利用
- SSH弱口令  
- MySQL弱口令  
- Redis未授权/弱口令  
- MSSQL弱口令  
- PostgreSQL弱口令  
- etcd未授权
- Kube API Server未授权
- Docker Daemon未授权

## Web PoC插件  
- PHP-CGI远程代码执行漏洞（CVE-2012-1823）
